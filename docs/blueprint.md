# **App Name**: Nova AI

## Core Features:

- Intelligent Conversational AI: Utilize a powerful generative AI tool to understand complex queries and provide intelligent, contextually relevant responses in both English and Arabic.
- Real-time Voice Interaction: Enable users to interact with the AI using speech-to-text input and receive responses via text-to-speech output, facilitating a natural, hands-free conversational flow within the web app.
- Persistent Chat History: Store and allow users to revisit past conversation sessions securely. This feature leverages Firestore for reliable and scalable data persistence.
- Multi-language Interface: Provide a user interface that can seamlessly switch between English and Arabic, including translation of prompts and display of AI responses in the selected language.
- User Account & Authentication: Allow users to register and log in securely, enabling personalized experiences and private storage of chat histories with Firebase authentication and Firestore user profiles.

## Style Guidelines:

- Primary color: A vibrant, rich violet (#8A4DF2) that signifies intelligence and creativity, contrasting beautifully with the dark background.
- Background color: A very dark gray (#17151A) with a subtle violet tint, creating a deep, immersive canvas that enhances readability and aligns with the requested dark theme.
- Accent color: A vivid electric blue (#5F7EFF), analogous to the primary color, providing striking contrast and supporting gradient aesthetics for highlights and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and objective aesthetic, suitable for clear readability in an AI assistant application.
- Utilize sleek, minimalist line icons or elegantly filled icons to maintain a clean, premium, and professional visual identity consistent with top-tier applications.
- Implement a 'glassmorphism' inspired or minimal aesthetic with distinctly rounded components throughout the interface to convey a modern and soft user experience.
- Integrate smooth, subtle animations for transitions, loading states, and interactive feedback, enhancing the premium feel and ensuring fluid navigation as requested.
